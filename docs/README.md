# Dokumentacja szablonu

Dokumentacja jest przeznaczona dla studentów przygotowujących prace
inżynierskie, magisterskie i doktorskie oraz dla osób utrzymujących szablon.

## Od czego zacząć

1. [Poradnik studenta](poradnik-studenta.md) — planowanie, pisanie i organizacja pracy.
2. [Przykłady LaTeX-a](latex-przyklady.md) — gotowe konstrukcje do wykorzystania.
3. [Lista kontrolna](lista-kontrolna.md) — sprawdzenie pracy przed oddaniem.

Krótka instrukcja instalacji i kompilacji pozostaje w głównym pliku
[`README.md`](../README.md). Zasady organizacji materiałów rozdziałów opisano
w [`chapters/README.md`](../chapters/README.md).

## Zasada redakcyjna

Poradnik wyjaśnia zarówno sposób pisania pracy, jak i użycie LaTeX-a. Dłuższe
instrukcje należy umieszczać tutaj, a w plikach rozdziałów pozostawiać tylko
krótkie komentarze odsyłające do odpowiedniej części dokumentacji. Dzięki temu
tekst instruktażowy nie trafi przypadkowo do końcowego PDF.
